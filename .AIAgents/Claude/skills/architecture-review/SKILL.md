# Architecture Review

## When to use

Use this skill for design reviews, tradeoff decisions, and risk analysis.

## Workflow

1. Read architecture context and constraints.
2. Evaluate options and tradeoffs.
3. Identify top risks with mitigation plans.
4. Define acceptance checks.

## Output

- Architecture review memo.
- Risk matrix with mitigation actions.
