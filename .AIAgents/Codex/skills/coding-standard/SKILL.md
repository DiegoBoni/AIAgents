# Coding Standard

## When to use

Use this skill for implementation tasks that require predictable quality checks.

## Workflow

1. Read local conventions and constraints.
2. Implement minimal, testable changes.
3. Run lint and tests.
4. Summarize behavioral impact and validation results.

## Output

- Code change aligned with project standards.
- Validation evidence.
